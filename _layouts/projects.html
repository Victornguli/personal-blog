Job Scraper CLI tool

Skraped is a cli tool that scrapes and aggregates job posts from several local job sites and boards. I built this tool so that I could easily sift through several job posts from a configured set of job sources and aggragate these into manageable csv files. Through a set of configurations, options and filters such as search keywords, blacklisted companies, duplicate checks, etc can be added to improve the aggregation process.

The core stack I used for this project is Python 3.6, BeautifulSoup library for parsing the scraped html, requests library for sending http requests to the scraped sources among other python libraries. The tool is also package ready and can be installed as a Python3 package and run on the terminal as such.
